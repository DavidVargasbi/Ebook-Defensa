# eBooks Ejecutivos · BI Consulting Lab

Sitio estático con las 6 guías ejecutivas sectoriales. Sin build, solo HTML.

## Estructura (todo en la raíz)
- `index.html` — portada con acceso a los 6 eBooks
- `defensa.html` — Defensa y Policía
- `salud.html` — Salud (IPS, EPS, clínicas, hospitales)
- `movilidad.html` — Movilidad y Servicios Públicos
- `contraloria.html` — Control Institucional y Justicia
- `economia.html` — Economía, Productividad y Vivienda
- `educativo.html` — Sector Educativo
- `vercel.json` — URLs limpias (`/defensa` en lugar de `/defensa.html`)

## Despliegue en Vercel
1. Sube **estos archivos a la raíz** del repositorio (no dentro de una subcarpeta).
2. Vercel → Add New → Project → Import del repo.
3. Framework Preset: **Other** · Build Command: *(vacío)* · Output Directory: *(vacío)*.
4. Deploy.
